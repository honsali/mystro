# Astro Rules

## First step

Read:

- `NEW_ARCHITECTURE_SPEC.md`

## Current philosophy

A doctrine is a hardcoded knowledge module.

Rules belong inside the doctrine module that exposes them.

The engine does not maintain a universal astrology rule schema and does not mark absent concepts as unsupported inside doctrine reports.
