Do not reinvent these algorithms from memory if `ASTRO_RULES.md` already records them. Read it first, then implement or refine.

Prioritize matching:
1. formulas
2. whole-sign house assignment method
3. sign-relationship logic

Only after that, investigate small degree differences.
